Что такое C# и зачем он вам

C# — это язык программирования, на котором можно делать реальные продукты, а не только учебные задачки. В этом модуле вы будете создавать штуки, которыми можно пользоваться: мини-игры, приложения с кнопками, таймеры, заметки, простые «продукты» под Windows.

Важная мысль: язык программирования — это способ объяснить компьютеру действия очень точно. Компьютер не угадывает и не «понимает намёки»: он выполняет то, что написано. Поэтому в программировании ценится ясность и порядок.

Почему именно C# для вашей группы:

  • Он логичный и «честный»: если написано неправильно — покажет ошибку
  • Он популярный и прикладной: навыки реально пригодятся дальше (и в проектах, и в будущем обучении)
  • Он отлично работает с Visual Studio: это один из самых удобных комплектов «язык + среда разработки»
🎵

Короткий факт: название C# читается «си-шарп», где sharp — музыкальный знак ♯ (как «прокачанный уровень»).

2

Из чего состоит «магия»: C#, .NET и Visual Studio

Чтобы дети не путались, сразу закрепи простую модель:

🔤 C#

  • Это язык (как мы пишем команды)

⚙️ .NET

  • Это платформа и набор готовых возможностей (готовые детали и суперсилы)

🏭 Visual Studio

  • Это мастерская, где всё создаётся (пишем, собираем, запускаем, находим ошибки)

Если объяснять совсем на пальцах — вы хотите сделать «приложение». Вам нужен:

  • язык, чтобы описать логику — C#
  • набор готовых функций, чтобы не изобретать всё заново — .NET
  • инструмент, где удобно всё собрать в проект и запускать — Visual Studio
3

Что такое .NET «по-человечески»

.NET нужен, чтобы не писать всё с нуля. Он даёт готовые решения:

  • вывод текста и работа с вводом
  • работа с файлами (сохранить/загрузить)
  • окна, кнопки, формы (это мы будем делать позже в WinForms)
  • таймеры, работа со временем
  • изображения, простая графика
  • и много чего ещё
🧩

Детям важнее всего понять: мы будем собирать проекты как конструктор, используя готовые детали .NET. Это сильно ускоряет разработку и делает «похоже на реальную работу».

4

Visual Studio: что это и что в ней важно

Visual Studio — это среда разработки (IDE). Это не просто «редактор текста», а комплект, где:

  • создают проект
  • пишут код
  • запускают программу
  • видят ошибки
  • чинят ошибки
  • и постепенно делают приложение всё больше и красивее
💡

Главная привычка новичка: если что-то не получается — не паника, а «смотрим, что пишет Error List». Это не наказание, а подсказка.

5

Как из текста получается работающая программа

Когда программист пишет программу, сначала это просто текст — исходный код. Чтобы этот текст стал «работающим приложением», происходит сборка (build / компиляция). Сборка — это проверка и превращение кода в то, что компьютер может выполнить.

Ошибки бывают двух типов (и это супер-база на весь модуль):

🔴 Ошибки компиляции

  • Программа не запускается вообще, потому что написано «не по правилам языка»
  • Как если в рецепте пропущены шаги — повар не начнёт

🟡 Ошибки логики

  • Программа запускается, но делает не то, что задумано
  • Как если рецепт выполнен, но ты добавил соль вместо сахара
✅

Нормализуй для детей: ошибки — это часть разработки. Программисты не «пишут идеально с первого раза», они пишут → проверяют → исправляют.

И главное: каждый урок будет заканчиваться чем-то, что можно показать: «вот моя программа, она делает X».

Глава 4 — Типы данных и переменные

Переменная — это коробка с именем, в которой хранится значение. В C# каждая переменная должна иметь тип данных.

Основные типы:

Тип Что хранит Пример
string текст string name = "Макс";
int целое число int age = 13;
bool правда / ложь bool isWin = true;
📄 Примеры 
string name = "Макс";
int age = 13;
bool isWin = true;
⚙️

Глава 5 — Работа с переменными

Добавить в код:

📄 Program.cs 
string name = "Артём";
int age = 14;

Console.WriteLine(name);
Console.WriteLine(age);
💬

Объяснение: переменные выводятся так же, как обычный текст.

Соединение текста и переменных:

📄 Program.cs 
Console.WriteLine("Меня зовут " + name);
Console.WriteLine("Мне " + age + " лет");
💬

Объяснение: знак + соединяет данные.

➕

Глава 6 — Можно ли сложить типы данных? Здесь задаём максимум вопросов детям

Число + число — обычная математика:

📄 Результат: 8 
int a = 5;
int b = 3;
Console.WriteLine(a + b);

Текст + текст — соединение:

📄 Результат: Артём Иванов 
string firstName = "Артём";
string lastName = " Иванов";
Console.WriteLine(firstName + lastName);

Текст + число — число превращается в текст:

📄 Результат: Мне 14 лет 
int age = 14;
Console.WriteLine("Мне " + age + " лет");

Число + текст — тоже соединение, а не сложение:

📄 Результат: 105 
int number = 10;
string text = "5";
Console.WriteLine(number + text);
🔢

Глава 7 — Настоящее сложение чисел

Чтобы сложить как числа, нужно преобразование:

📄 Результат: 15 
int number = 10;
string text = "5";
int result = number + Convert.ToInt32(text);
Console.WriteLine(result);
💬

Объяснение: Convert.ToInt32() превращает текст в число.

Глава 8 — Закрепление

🎯

Задача: создать программу, которая спрашивает имя, спрашивает возраст и выводит сообщение: «Тебе X лет — ты начинающий программист».

Добавить в код:

📄 Program.cs — Закрепление 
Console.WriteLine("Как тебя зовут?");
string userName = Console.ReadLine();

Console.WriteLine("Тебе " + userAge + " лет --- ты начинающий программист");

Если нужно вывести имя:

Console.WriteLine(userName + ", тебе " + userAge + " лет --- ты начинающий программист");
✅

Проверить: в конце строк стоит ;, число не в кавычках, используется Convert.ToInt32().

Что такое if

if — это условие. Программа проверяет выражение в скобках. Если условие верное (true) — выполняется код в { }. Если неверное (false) — программа идёт дальше.

Пример:

📄 Пример if 
int number = 5;

if (number > 3)
{
    Console.WriteLine("Больше трёх");
}

= присвоение

  • Записать значение в переменную
  • int iq = 100;

== сравнение

  • Проверить: равны ли два значения?
  • if (iq == 100)
🧠

Мини-проект — Шуточный генератор IQ

📄 Program.cs — Генератор IQ 
Console.WriteLine("Как тебя зовут?");
string name = Console.ReadLine();

Random random = new Random();
int iq = random.Next(60, 181);

Console.WriteLine(name + ", твой программистский IQ: " + iq);

if (iq < 90)
{
    Console.WriteLine("Начальный уровень 😄");
}

if (iq >= 90 && iq < 130)
{
    Console.WriteLine("Хороший уровень 🎮");
}

if (iq >= 130)
{
    Console.WriteLine("Ты почти гений 🚀");
}

if (iq == 180)
{
    Console.WriteLine("Максимум! Это легенда 🔥");
}
